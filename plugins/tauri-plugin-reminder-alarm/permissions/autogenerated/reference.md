## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-schedule-exact-alarm`
- `allow-cancel-alarm`
- `allow-can-schedule-exact-alarms`
- `allow-request-exact-alarm-permission`
- `allow-ensure-notification-permission`
