## Full description

FocusNook помогает держать планы на день рядом, не превращая их в отдельный
сложный проект. Добавляйте задачи, отмечайте прогресс, сохраняйте текстовые и
голосовые заметки и настраивайте напоминания.

Данные прежде всего хранятся на устройстве. По желанию можно создать аккаунт
FocusNook и синхронизировать зашифрованные данные между Android и Windows через
сервер FocusNook. Для локальной работы аккаунт не нужен.

Возможности первой версии:

- задачи на сегодня со статусами и прогрессом;
- текстовые и голосовые заметки;
- системные напоминания, включая восстановление после перезагрузки;
- несколько локальных профилей;
- необязательная синхронизация Android и Windows;
- удаление аккаунта и серверных данных из приложения.

FocusNook не содержит рекламы и покупок внутри приложения.
