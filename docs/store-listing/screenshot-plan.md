# Store screenshot capture plan

Capture real screens from the signed release candidate on a physical phone.
Do not use mockups or show functionality absent from the candidate.

Use portrait `1080 x 1920` PNG images (9:16), consistent light or dark theme,
Russian UI for RuStore, and English plus Russian localized sets for AppGallery.
Avoid personal email addresses, real plans, notification shade content, debug
overlays, and device navigation chrome.

Required sequence:

1. Today view with three neutral sample tasks and visible progress states.
2. Quick creation/editing of a task.
3. Notes view with a neutral text note and voice-note action.
4. Reminders view with two future reminders.
5. Settings showing local-first mode and optional FocusNook sync.
6. Connected-account settings showing privacy link and account deletion entry.

Before upload, compare every image with the final build, confirm readable text,
and record device/model/build in the release evidence. RuStore requires at least
one phone screenshot; this plan intentionally provides six.
