## Introduction

FocusNook keeps today's plans close without turning them into a complex project.
Create tasks, track progress, keep text or voice notes, and schedule reminders.

Your data is local first. If you choose to create a FocusNook account, encrypted
planner data can sync between Android and Windows through the FocusNook server.
An account is not required for local use.

The first release includes today's tasks, progress states, text and voice notes,
system reminders that survive reboot, multiple local profiles, optional Android
and Windows sync, and in-app deletion of the account and server data. FocusNook
contains no advertising or in-app purchases.
